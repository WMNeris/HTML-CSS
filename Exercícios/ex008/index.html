Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em negrito usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase temos um termo em itálico usando a tag <em> (semântica).

Texto Marcado

Podemos marcar uma parte do texto como esta usando a tag <mark>

Para mudar a cor de todos os marks cria-se um seletor dentro da tag <style> em <head>.

Texto Prande e Pequeno

Para texto com letras grandes usava-se a tag <big> (Que se tornou obsoleta).

Neste exemplo temos um texto pequeno usando a tag <small>("define um texto menor (como direitos autorais e outros comentários secundários)."

Texto Excluído / Deletado

Para um techo excluído usa-se a tag <del>.

Texto Inserido

Para inserir um texto como este exemplo usamos a tag <ins>.

Também podemos usar a tag <u> (não semântica. É recomendado usar a tag <ins>).

Texto Sobrescrito

Para sobrescrever como no termo a seguir: x20+3 usamos a tag <sup>.

Texto Subscrito

Para subscrever como no exemplo a seguir: H2O, usa-se a tag <sub>.
